## ✨ 特性

- 👍 **非侵入式** 尽可能提供原生 Cesium 的开发体验，并尽量减少对项目的入侵，这使得你能低成本的在项目中使用它，甚至只是使用它提供的某个工具函数。
- 📝 **TypeScript** 基于 TS 编写，为 JS 和 TS 用户提供同样的类型支持。
- 🚀 **现代** CesiumUse 旨在使用简洁易懂的 API，提供现代化的开发体验。
